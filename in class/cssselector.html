CSS Selectors

Applying CSS to Different Parts of HTML

  1. Element Selector: Targets elements based on their HTML tag name.
  2. Class Selector: Targets elements based on the value of the class attribute.
  3. ID Selector: Targets elements based on the value of the id attribute.
  4. Attribute Selector: Targets elements based on their attributes and values.
  5. Universal Selector: Targets all elements.
